Countdown: The goal is to get as many points as you can before the time runs out. Each correct answer grants one point. You will get bonus time as you answer more questions correctly in a row. When there is a wrong answer, the combo bar is depleted, and you lose 10 seconds.

Frenzy: The goal is to answer as many questions correctly in a row. With one wrong answer, the game ends. However, you can choose to skip one question using the skip button.
